## 🚀 كيفية التشغيل

1. قم بتنزيل الملفات
2. افتح ملف `index.html` في متصفحك
3. ابدأ باستخدام الآلة الحاسبة!

## 💻 كيفية الاستخدام

### استخدام الفأرة:
- انقر على الأرقام والعملية المطلوبة
- انقر على `=` للحصول على النتيجة
- `C` لمسح كل شيء
- `⌫` لحذف آخر رقم

### استخدام لوحة المفاتيح:
- **الأرقام**: 0-9
- **العمليات**: +, -, *, /
- **مسح**: Escape
- **حذف**: Backspace
- **نتيجة**: Enter أو =
